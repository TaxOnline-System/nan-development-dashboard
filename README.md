# nan-development

ระบบ Dashboard สำหรับติดตามและวิเคราะห์ความสอดคล้องของโครงการจังหวัดน่านกับแผนพัฒนาจังหวัด แผนงานหลัก ตัวชี้วัด ค่าเป้าหมาย และ SDGs

## สิ่งที่มีในเวอร์ชันนี้

- ตัดเมนู Export ออกทั้งหมด
- ใช้ชื่อระบบ `nan-development`
- ใส่โลโก้ AIAT และ RMUTL
- Dashboard ภาพรวมจำนวนโครงการ งบประมาณ และระดับความสอดคล้อง
- กราฟจำนวนโครงการ งบประมาณตามกลุ่มเป็นเปอร์เซ็นต์ และ SDGs
- ตารางโครงการพร้อมค้นหาและกรองข้อมูล
- หน้ารายละเอียดโครงการพร้อมข้อมูลสำหรับเจ้าของโครงการ
- Checklist รายโครงการก่อนปรับแก้
- ข้อเสนอแนะและตัวอย่างข้อความ Outcome ที่นำไปปรับใช้ได้

## วิธีอัปโหลดขึ้น GitHub Pages

1. แตกไฟล์ ZIP นี้
2. สร้าง Repository ใหม่บน GitHub ชื่อ `nan-development`
3. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น Repository
4. ไปที่ Settings > Pages
5. เลือก Branch: `main` และ Folder: `/root`
6. กด Save
7. รอ GitHub Pages สร้าง URL

## แก้ไขข้อมูล

แก้ไขไฟล์ `data/projects.json` แล้ว Commit ขึ้น GitHub หน้า Dashboard จะอัปเดตตามข้อมูลใหม่

จำนวนข้อมูลในชุดนี้: 112 โครงการ
