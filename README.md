# nan-development

ระบบ Dashboard สำหรับติดตามและวิเคราะห์ความสอดคล้องของโครงการจังหวัดน่านกับแผนพัฒนาจังหวัด ตัวชี้วัด แผนงานหลัก และ SDGs

## จุดประสงค์

ให้เจ้าของโครงการเข้ามาตรวจสอบข้อมูลของตนเอง เห็นระดับความสอดคล้อง เห็น SDGs ที่เกี่ยวข้อง และเห็นข้อเสนอแนะสำหรับปรับแก้โครงการให้เชื่อมโยงกับแผนพัฒนาจังหวัดได้ชัดเจนขึ้น

## เวอร์ชันนี้ปรับปรุงแล้ว

- ตัดเมนู Export ออกทั้งหมด
- ปรับขนาดฟอนต์ใหม่ให้อ่านง่าย ไม่ใหญ่เกินไป
- เพิ่มรายละเอียดแผนงานหลัก ตัวชี้วัด เกณฑ์ประเมิน Output Outcome และข้อเสนอแนะรายโครงการ
- เพิ่ม Checklist สำหรับเจ้าของโครงการ
- แสดงงบประมาณเป็นจำนวนเงินและร้อยละ
- พร้อมใช้งานบน GitHub Pages

## ข้อมูลในระบบ

- จำนวนโครงการ: 112 รายการ
- งบประมาณรวม: 1,067,920,918.84 บาท
- กลุ่มข้อมูล: สังคม, ท่องเที่ยว, ทรัพยากร, การค้า, เกษตร

## วิธีอัปโหลดขึ้น GitHub Pages

1. แตกไฟล์ ZIP
2. สร้าง Repository ชื่อ `nan-development`
3. อัปโหลดไฟล์ทั้งหมดขึ้น Repository
4. ไปที่ Settings > Pages
5. เลือก Deploy from branch
6. เลือก Branch `main` และ Folder `/root`
7. กด Save

ระบบจะเปิดใช้งานได้ผ่าน URL ของ GitHub Pages
